## わかったこと

- シンボルと文字列の違い
  - シンボルは内部的には整数である
- ハッシュ、js の連想配列と同等
- brake と return の違い
  - brake は繰り返し処理からの脱出であり、return はメソッドからの脱出
- next 繰り返し処理を中断し、次の繰り返し処理に進む
- redo 繰り返し処理の初めに戻る
- ruby における catch 文は例外処理ではない
- ruby で for 文は推奨されない理由は each と違いブロックがないためスコープがグローバルになりバクの温床になる
